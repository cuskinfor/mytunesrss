The web application

When the server has been started, you can access the web application with an browser. Your first test should be to use a browser on the same machine you are running MyTunesRSS with the address "http://127.0.0.1:8080". Use the port you have configured in the server GUI. If this works, MyTunesRSS is up and running. If you have problems accessing the web application from elsewhere, you have network problems not related to MyTunesRSS in particluar. Please check your network and firewall in this case.
