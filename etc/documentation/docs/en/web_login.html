Login

Da die Webanwendung durch ein Kennwort vor unbefugtem Zugriff geschützt ist, gelangen Sie mit Ihrem Browser als erstes auf die Loginmaske. Hier müssen Sie das Kennwort eingeben, das sie in der MyTunessRSS Konfiguration vergeben haben. Wenn Sie die Option "eingeloggt bleiben" aktivierern wird ein Cookie im Browser gesetzt und sie werden beim nächsten Zugriff automatisch eingeloggt. Verwenden Sie diese Option also nicht, wenn Sie von öffentlichen Browsern aus auf Ihre Musikbibliothek zugreifen.
