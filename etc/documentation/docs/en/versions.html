Version history

Version 2.2.3

August 10, 2006
  • Button for deleting and recreating the database
  • Display artwork on track info page (RSS item link)
  • Fixed mime type bug when downloading tracks
  • Fixed bug when no address was found in server info

Version 2.2.2

August 8, 2006
  • Option to disable MP3 artwork in RSS feeds
  • Stability improvements for MP3 artwork support

Version 2.2.1

August 6, 2006
  • Added missing library

Version 2.2

August 6, 2006
  • Download of multiple titles in ZIP archive
  • Option to ignore timestamps on database updates
  • More information in server status window
  • Album art from MP3s in RSS feed
  • Bugfixes

Version 2.1

June 25, 2006
  • Download link for individual tracks
  • Database auto-update in minutes instead of seconds
  • Older iTunes versions should work
  • Display of local/external address in dialog
  • Error sound when requesting missing files
  • Bugfixes

Version 2.0.1

June 10, 2006
  • Fixed bug which caused most RSS feeds to fail on PSP
  • Fixed bug which caused database failure with some artist/album names
  • Database stored in special folder under user home

Version 2.0

June 7, 2006
  • All file types from the iTunes library are used
  • Browse the library by album or artist
  • Manage own playlists with MyTunesRSS
  • Create M3U playlists
  • Automatic server start when starting MyTunesRSS
  • Automatic updates from iTunes during runtime
  • On Windows systems the application can be minimized to the system tray
  • User interface comletely redesigned

Version 1.2

May 7, 2006
  • Optionally limit size of an RSS feed
  • Compose RSS feed from multiple search results
  • Automatic checking for updates

Version 1.1

May 1, 2006
  • Support for unprotected AAC files
  • Create RSS directly from iTunes playlists
  • Protect web pages with password
  • Less memory consumption
  • Improved user interfaces
  • Easier to use
  • Overall stability improvements

Version 1.0.1

April 5, 2006
  • Missing files do not crash application
  • Log files for better maintenance

Version 1.0

April 2, 2006
  • First official release
