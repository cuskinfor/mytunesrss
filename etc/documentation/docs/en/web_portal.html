Portal

The portal is the starting point for MyTunesRSS. You can see all available playlists, which are the ones from iTunes (some are not used, e.g. the party shuffle) and the MyTunesRSS playlists. The MyTunesRSS playlists are the ones you create using the MyTunesRSS web application. The different types have different icons.

You can reach the settings and you can log out here. When logging out, the cookie that might have been set using the option to stay logged in is discarded. So you get really logged off and you will have to enter your password again. You can start browsing the library for artists and albums and you can open the playlist manager to work with your MyTunesRSS playlists.

The search feature is useful in case you know what you are looking for. You can enter any number of search terms separated by space. All search terms must be found in the name, the album or the artist of a track to get a match. For example searching for "madonna music" will find all tracks by Madonna which have the word "music" in either the album or the track name. The search is case-insensitive.

The track count for each playlist is displayed. A click on the count displays the tracks. At the end of each row you can see the icons for creating an M3U playlist or an RRS feed from the playlist tracks. You can also download a ZIP archive with all titles of a playlist by clicking the download icon.
