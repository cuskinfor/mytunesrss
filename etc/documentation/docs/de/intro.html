Einführung

Mit MyTunesRSS können Sie aus der Musik in Ihrer iTunes Musikbibliothek einen RSS Feed erstellen. Die Titel des Feeds können dann von jedem Gerät abgespielt werden, dass einen Browser besitzt und RSS Feeds unterstützt. Ein solches Gerät ist beispielsweise die Sony Playstation Portable mit einer entsprechenden Firmware.

MyTunesRSS besteht aus einer kleinen Oberfläche zur Konfiguration und zum Starten eines Servers. Wenn der Server gestartet ist, kann mit einem Webbrowser auf eine Webanwendung zugegriffen werden. Mit dem Browser können Sie Ihre iTunes Musikbibliothek durchsuchen und den RSS Feed erzeugen.

MyTunesRSS ist Shareware. Sie können den Basisumfang der Anwendung kostenlos nutzen. Wenn Sie die erweiterten Funktionen nutzen möchten, so müssen Sie einen Registrierungsschlüssel erwerben. In der Basisversion können Sie Ihre Bibliothek nach Künster und Album durchsuchen und aus den Ergebnissen einen RSS Feed zusammenstellen. Es wird nur das MP3 Format unterstützt. Nach einer Registrierung können Sie zusätzlich ungeschützte AAC Dateien nutzen, direkt aus Ihren iTunes Wiedergabelisten einen RSS Feed erzeugen und die Webanwendung mit einem Kennwort schützen.
