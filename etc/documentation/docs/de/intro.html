Einführung

MyTunesRSS ist eine Anwendung, mit deren Hilfe Sie von beliebigen Rechnern über das Netzwerk auf Ihre iTunes Bibliothek zugreifen können. Sie können mit Hilfe eines Webbrowsers Ihre Bibliothek durchsuchen, vorhandene Playlisten nutzen oder neue Playlisten erstellen. Neue Playlisten sind nur in MyTunesRSS verfügbar, nicht in iTunes. Aus all diesen Daten können Sie aus dem Webbrowser heraus RSS Feeds oder M3U Playlisten erstellen und mit Hilfe entsprechender Programme nutzen. So können Sie Ihre Musik beispielsweise auch auf modernen Endgeräten wie der Playstation Portable nutzen.

MyTunesRSS besteht aus einer Oberfläche zur Konfiguration und Steuerung des Servers und einer Webapplikation. Sobald der Server gestartet ist, können Sie auf die Webapplikation zugreifen. Der Zugriff auf die Webanwendung ist durch ein Kennwort gesichert.

Sie können Ihre Bibliothek nach Alben, Künstlern oder einzelnen Titeln durchsuchen. Neben den in iTunes vorhandenen Playlisten können Sie aus dem Daten auch eigene Playlisten zusammenstellen. Aus jeder Playliste, einem kompletten Album oder allen Titeln eines Künstlers können Sie M3U Playlisten und RSS Feeds erzeugen. Sie können einzelne Titel auch einfach herunterladen, zum Beispiel um sie ohne eine USB-Verbindung auf Ihre Playstation Portable zu kopieren. Anwendungen wie VLC können auch Streams von Videos aus Ihrer Bibliothek wiedergeben.
